#### 领导术 城堡、教廷、中立
#### 幸运术 壁垒、禁林、花仙堡
#### 恐吓术 墓园、梦境、死亡谷
#### 巫蛊术 要塞、沙堡、地下城
#### 火系魔法 地狱、堡垒、工厂
#### 水系魔法 港口、峡湾、冰狱
#### 干扰术 据点、元素
#### 侦察术 塔楼、贵族

#### 智慧术 力量+1；力量+2；力量+3
